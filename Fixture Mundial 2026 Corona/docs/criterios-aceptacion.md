# Criterios de aceptacion

El MVP se considera listo para publicar cuando se cumpla:

- Registro guarda participantes en Google Sheets.
- Email duplicado es rechazado.
- Admin puede crear y editar partidos.
- Admin puede cargar resultados.
- Predicciones se crean o actualizan antes del cierre.
- Predicciones quedan bloqueadas despues del cierre.
- Resultado exacto suma 5 puntos.
- Ganador correcto o empate correcto suma 2 puntos.
- No hay doble conteo cuando hay resultado exacto.
- Ranking individual ordena por puntaje y desempates.
- Ranking por area usa promedio de los mejores 5.
- Reglamento y premios se muestran con fallback si no hay configuracion.
- Frontend funciona en celular y escritorio.
- README explica configuracion, despliegue y pruebas.
