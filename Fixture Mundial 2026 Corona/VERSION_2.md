# Version 2.0

Mejoras fuera del MVP para no comprometer el plazo de publicacion.

## Seguridad y usuarios

- Login seguro con usuario y contrasena.
- OAuth corporativo.
- Roles: participante, administrador, auditor.
- Recuperacion de contrasena.

## Backend productivo

- Migracion a PHP + MySQL.
- API propia versionada.
- Separacion test/prod.
- Logs persistentes y auditoria de cambios.

## Integraciones

- API FIFA u origen oficial de fixtures/resultados.
- Emails automaticos.
- WhatsApp.
- Notificaciones push.

## Experiencia y analitica

- PWA.
- Dashboard avanzado.
- Estadisticas por participante, area, fase y partido.
- Historial de cambios.
- Comentarios o muro interno.
- Gamificacion avanzada con insignias.
- Multiidioma.
