# AGENTS.md - Ingenio La Corona / Sistemas

## Contexto
Este repositorio contiene desarrollos internos del área de Sistemas del Ingenio La Corona.

## Reglas generales
- No modificar archivos fuera del alcance solicitado.
- No usar credenciales reales.
- No inventar tablas, columnas ni endpoints.
- Antes de cambiar código, explicar brevemente el plan.
- Priorizar soluciones simples, mantenibles y documentadas.
- Todo cambio debe incluir una forma de prueba.
- No ejecutar acciones sobre producción.
- No modificar lógica de ERP, PLC, SCADA o base productiva sin autorización explícita.

## Tecnologías habituales
- Node-RED
- n8n
- JavaScript
- Python
- PHP
- MySQL
- SQL Server 2008 R2
- Docker
- APIs REST
- ERP Calipso
- Power BI / dashboards web

## Criterio de entrega
Cada tarea debe devolver:
1. Archivos modificados.
2. Resumen del cambio.
3. Cómo probarlo.
4. Riesgos o supuestos.
